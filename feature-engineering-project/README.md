Feature Engineering Project (Google Colab ready)

Files included:
- feature_engineering_colab.ipynb : Colab-friendly notebook (load data from data/ folder or upload in Colab).
- theory_report.docx : Word report with embedded generated plots.
- data/WA_Fn-UseC_-HR-Employee-Attrition.csv : Original dataset.
- data/feature_engineered_ibm_hr.csv : Processed dataset.
- images/ : Plots used in report.

How to use in Colab:
1. Upload the ZIP to your Google Drive or directly open the notebook in Colab (File -> Upload notebook).
2. If you upload the notebook only, also upload the CSV using 'files.upload()' or mount Google Drive.
3. Run cells sequentially.
